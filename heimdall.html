About

Heimdall Application Dashboard is a dashboard for all your web applications.

Install

Step 1 - Download Heimdall from GitHub

Download Heimdall from GitHub

RELEASE=$(curl -sX GET "https://api.github.com/repos/linuxserver/Heimdall/releases/latest" | awk '/tag_name/{print $4;exit}' FS='[""]'); echo $RELEASE &&\
curl --silent -o ${RELEASE}.tar.gz -L "https://github.com/linuxserver/Heimdall/archive/${RELEASE}.tar.gz"

Unzip the tar

tar xvzf 2.2.2.tar.gz

 

Step 2 - Install PHP modules

Install PHP models 

sudo apt install php-sqlite3 php-zip

 

Step 3 - Create Heimdall systemd service

Create heimdall systemd service

sudo nano /etc/systemd/system/heimdall.service

Add the following content to the file

[Unit]
Description=Heimdall
After=network.target

[Service]
Restart=always
RestartSec=5
Type=simple
WorkingDirectory=/home/lgtm/Heimdall-2.2.2
ExecStart="/usr/bin/php" artisan serve --port 7889 --host 0.0.0.0
TimeoutStopSec=30

[Install]
WantedBy=multi-user.target

 

Step 4 - Enable and start the Heimdall service

Enable to service to start on boot.

sudo systemctl enable  heimdall.service

Start the Heimdall service

sudo systemctl start heimdall.service

 

Step 5 - Check to see if service had started 

Check the status of the heimdall service and see if it success fully started

sudo systemctl status heimdall.service

 

Step 6 - Visit your Heimdall dashboard

Check out your dashboard at your localhost address or use your machines hostname/ip

http://127.0.0.1:7889/
